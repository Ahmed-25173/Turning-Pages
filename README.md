# Turning Pages -- Online Bookstore Website

## Project Overview

**Turning Pages** is a fully responsive static online bookstore website
built using HTML5 and CSS3.

The project simulates a real-world e-commerce platform where users can:

-   Browse books by category
-   View detailed book pages
-   Add items to a shopping cart
-   Proceed to a checkout page

It demonstrates strong front-end fundamentals, clean UI design, and
structured multi-page navigation.

## Installation & Setup


1.  Clone the repository:

{=html}

    git clone https://github.com/Ahmed-25173/Turning-Pages.git

2.  Navigate into the project folder:

{=html}

    cd Turning-Pages

3.  Open `home.html` in your browser.

No additional dependencies are required.
